# VinisTech.in – Digital Innovation & AI Solutions

VinisTech.in is a modern, responsive portfolio website showcasing interactive demos, free tools, and a blog. Built with HTML5, CSS3 (glassmorphism/neumorphism), and vanilla JavaScript. Uses a build‑time assembly system for maintainability.

## Features

- Fully responsive, glassmorphic design with dark/light mode.
- Mega‑menu, floating chatbot, lead capture.
- Dynamic demo hub (`demo.html`) – automatically lists all subfolders in `demos/`.
- Dynamic tools hub (`tools.html`) – automatically lists all subfolders in `tools/`.
- Blog with modal popup for full articles.
- Modular header/footer injected via build script.
- Optimised images and lazy loading.

## Tech Stack

- HTML5, CSS3 (custom properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Node.js build scripts (for assembly and path fixing)
- Vercel for deployment
